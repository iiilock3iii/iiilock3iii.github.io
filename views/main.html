Senior Project

Lovesac Sactional Program

This is a senior project that I worked on for my brother who works at Lovesac. The premise of the project is, each employee has a handbook with 60 different couch configurations, depending on how many bases and sides you own. So the purpose of the project is to let a user enter in how many bases and sides they want, and then the page will filter to the possible couches they can create.
